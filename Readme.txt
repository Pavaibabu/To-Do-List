Simple To-Do List Web App

A simple to-do list web app that allows you to add tasks, mark them as completed, and remove them. The app uses HTML, CSS, and JavaScript for the frontend, and utilizes the localStorage API to store tasks locally, ensuring your tasks are persisted even after refreshing the page.

Features
Add new tasks to the list by typing in the input field and clicking the "Add" button.
Mark tasks as completed by clicking on the task.
Remove tasks by clicking the close button (✖) associated with each task.
Tasks are saved locally using the localStorage API, so they remain even after refreshing the page.
Getting Started
To run the web app, follow these steps:

Clone this repository to your local machine using git clone https://github.com/Pavaibabu/To-Do-List.git

Open the index.html file in your preferred web browser.

Usage
Type your task in the input field and click the "Add" button to add it to the list.

Click on a task to mark it as completed. Click again to undo.

Click the close button (✖) to remove a task from the list.

Your tasks will be saved locally and will persist even after you refresh the page.

Code Structure
The web app's structure is as follows:

index.html: The main HTML file that defines the structure of the app.
styles.css: CSS file containing styles for the app's appearance.
script.js: JavaScript file containing the app's functionality, including adding, marking, and removing tasks, as well as using localStorage to save and load tasks.
